ABOUT PYTHON:

  • It is general purpose language.

  • It is developed by Guido Van Rossum in 1991.

  • It comes from an old BBC Tv comedy sketch series called Monty Python's Flying Circus.

  • By using python you can develop:

  1. Web Applications
  2. Desktop Applications
  3. Mobile Applications
  4. To develop Games

FEATURES OF PYTHON:

  1. It is simple and easy.

  2. Syntax is easy and it is open-source.

  3. It is dynamically typed programming language.

  4. It is embedded and extensible.

  5. It is interpreted programming language.

PYTHON RULES FOR IDENTIFIERS:

  1. Special symbols are not allowed.

  2. Identifiers are case-Sensitive.

  3. Identifiers should not start with digits.
